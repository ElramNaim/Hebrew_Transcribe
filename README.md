# Hebrew_Transcribe
## to start the backend:  
* download pipenv
* open cmd
* enter your project path from the cmd : 'cd/../yourCloneProject/'  
* type 'pipenv shell'  
* 'cd backend'  
* 'python manage.py runserver'  
## to start the frontend :  
* open a new cmd
* enter your project path from the cmd : 'cd/../yourCloneProject/'  
* 'cd frontend'  
* 'npm start'
